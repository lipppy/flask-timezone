Timezone information

API call:

  • {{ url_for('api.timezones', _external=True) }}
  • {{ url_for('api.timezones', _external=True) }}?lat={latitude}&lon={longitude}
  • {{ url_for('api.timezones', _external=True) }}?lat={latitude}&lon={longitude}&utctype=0
  • {{ url_for('api.timezones', _external=True) }}?lat={latitude}&lon={longitude}&utctype=1

Parameters:

  • without parameter or with unknown parameter(s): displays all available timezones
  • lat: latitude in EPSG:4326 coordinate reference system
  • lon: longitude in EPSG:4326 coordinate reference system
  • utctype: determine UTC timezone, can be 0 (from UTC timezone database) or 1 (calculate UTC timezones manually)

Examples of API calls:

  • {{ url_for('api.timezones', _external=True) }}?lat=48.19&lon=16.39 - returning timezone "Europe/Vienna"
  • {{ url_for('api.timezones', _external=True) }}?lat=47.49&lon=19.04 - Budapest, Hungary - returning timezone "Europe/Budapest"
  • {{ url_for('api.timezones', _external=True) }}?lat=34.02&lon=-118.49 - Los Angeles, USA - returning timezone "America/Los_Angeles"
  • {{ url_for('api.timezones', _external=True) }}?lat=-81.31&lon=-175.49 - Ross Ice, Antarctica - returning timezone "Unknown"
  • {{ url_for('api.timezones', _external=True) }}?lat=-81.31&lon=-175.49&utctype=0 - Ross Ice, Antarctica - returning timezone "UTC-12" (by UTC zones database)
  • {{ url_for('api.timezones', _external=True) }}?lat=1.75&lon=-157.3 - a little north from (1°) Kiribati, Christmas Island, Pacific Ocean - returning timezone "Pacific/Kiritimati"
  • {{ url_for('api.timezones', _external=True) }}?lat=2.75&lon=-157.3 - a little north from (1°) Kiribati, Christmas Island, Pacific Ocean - returning timezone "UTC+14" (from UTC zones data)
  • {{ url_for('api.timezones', _external=True) }}?lat=2.75&lon=-157.3&utctype=1 - a little north from (1°) Kiribati, Christmas Island, Pacific Ocean - returning timezone "UTC-10" (by manually calculated UTC zones)
